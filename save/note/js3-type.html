Null类型

Null类型是第二个只有一个值的数据类型，这个特殊的值是null。

从逻辑角度来看，null值表示一个空对象指针，而这也正是使用typeof操作符检测null时会返回"object"的原因，例如：var car = null;
如果定义的变量准备在将来用于保存对象，那么最好将该变量初始化为null而不是其他值。这样一来，只要直接检测null值就可以知道相应的变量是否已经保存了一个对象的引用了，例如：

  1. if(car != null){
  2. //对car对象执行某些操作
  3. }
null和undefined

实际上，undefined值是派生自null值的，因此ECMA-262规定对它们的相等性测试要返回true。

  1. var obj; //undefined 没有定义
  2. var obb = null; //引用了内存中一块空的空
  3. var bj = null==undefined; //true
  4. var bj = null===undefined; //false

尽管null和undefined有这样的关系，但它们的用途完全不同。无论在什么情况下都没有必要把一个变量的值显式地设置为undefined，可是同样的规则对null却不适用。换句话说，只要意在保存对象的变量还没有真正保存对象，就应该明确地让该变量保存null值。这样做不仅可以体现null作为空对象指针的惯例，而且也有助于进一步区分null和undefined。

虽然Boolean类型的字面量只有true和false两种,但ECMAScript中所有的值都与这两个Boolean值等价的值,要将一个值转换为其对应的Boolean值,可以使用转型函数Boolean();

Number类型

Number类型包含两种数值:整型和浮点型.

为了支持各种数值类型,ECMA-262定义了不同的数值字面量格式.

  • 整型 - int
    • 十进制 var = 250;
    • 八进制 - 前导必须是0,十进制输出
    • 十六进制 - 前面两位必须是0x,后面必须是(0~9,A~F)
  1. //八进制
  2. var = 070; // 56
  3. var = 079; // 79 无效八进制 自动解析为十进制.
  5. //十六进制
  6. var box = 0xA; // 10
  7. var box = 0x1f; // 31
  • 浮点类型 - float double
  1.  var box = 3.8; // 3.8
  2.  var box = 03.8; // flase 报错
  3.  var box = 3.80; // 3.8 后导的0去掉
  4.  var box = .8; // 0.8 有效但不推荐
  5.  var box = 8.; // 8 自动去掉./0

科学记数法
对于过大或者过小的数值,可以用科学计数法(e表示法).用e表示该数值的前面10的指数次幂.

  1. var box = 4.12e9; // 4120000000 4.12*10的9次方
  2. var box = 0.00000000412; // 4.12e-9

小数表示不精准
虽然浮点数值的最高精度是17位小数,但算术运算中可能会不精确.由于这个因素,做判断的时候一定要考虑到这个问题,case:

  1. alert(0.1+0.2); // 0.300000004 不精确 因此不要用浮点做精确运算

正无穷和负无穷
也可以通过NumberPOSITIVE_INFINITY和NumberNEGATIVE_INFINITY得到正无穷以及负无穷的值.

  1.  alert(Number.POSITIVE_INFINITY); // Infinity;
  2.  alert(Number.NEGATIVE_INFINITY); // -Infinity;

最大值最小值
浮点数值的范围在:NumberMIN_VELUE~NumberMAX_VALUE之间.

  1. alert(Number.MIN_VELUE); // 5e-324 最小值
  2. alert(Number.MAX_VALUE); // 1.797693***e+308 最大值

如果超过了最大值或者最小值:

  1. var box = 100e1000; // Infinity;
  2. var box - -100e1000; // -Infinity;

想要确定一个数值到底是否超过了规定范围,可以使用isFinite()函数,没有超过返回true,超过了返回flase.

  1. var box = 100e1000;
  2. alert(isFinite(box)); // flase

JS生成随机数

  • 从1开始 至 任意值
  1. parseInt(Math.random()*上限+1);
  • 从任意值开始 至 任意值
  1. parseInt(Math.random()*(上限-下限+1)+下限);

上面的公式使用了parseInt(), 因此要加1; 如果使用 Math.ceil() 则不需要加1, 俺习惯于这样写...

NaN

呵呵哒 本来要返回一个数字,但未返回数字

  1. var box = 0/0; // NaN
  2. var box = 12/0; // Infinity
  3. var box = 12/0*0; // NaN

得到NaN的值的方法; 任何与NaN运算的结果都是NaN,但NaN!=NaN

  1. alert(Number.NaN); // NaN 得到NaN的值
  2. var box = NaN+1; // NaN
  3. alert(NaN==NaN); // flase 不相等,不是一个数字,可能是除以0造成,可能是Infinity造成.

ECMA-262提供了isNaN()的函数,在收到一个值后,会尝试将这个值转化为数值.

  1. alert(isNaN(NaN)); // true
  2. alert(isNaN(25)); // flase,25是一个数值
  3. alert(isNaN("25")); // flase,"25"是一个字符串数值,可以转化为数值
  4. alert(isNaN("canlan")); // true,"canlan"不能转换为数值
  5. alert(isNaN(true)); // flase,true==1

使用数字运算需要注意的地方

  1. 'abc' - 10  // NaN
  2. NaN * 1 // NaN
  3. 0 / 0  // NaN
  4. 非0正数 / 0  // Infinity
  5. 非0负数 / 0  // -Infinity
  6. Infinity / Infinity  // NaN
  7. Infinity - Infinity   // NaN
  8. Infinity % Infinity   // NaN

isNaN()函数也适用于对象,在调用isNaN函数的过程中,首先会调用valueOf()方法,然后确定返回值是否能够转换成数值,如果不能,则基于这个返回值再调用toString()方法,再测试返回值.如果对象的toString方法能够返回值, 那么就不是NaN了

  1. var box = {};
  2. alert(isNaN(box)); // true
  4. var box = {
  5.     toString:function(){
  6.         return 123;
  7.     }
  8. }   // flase
  10. var box = {
  11.     toString:function(){
  12.         return "canlan";
  13.     }
  14. }  // true
转化数值

有3个函数可以把非数值转换为数值:Number()``parseInt()``parseFloat()

  • Number()
    Number()函数是转型函数,可以用于任何数据类型,而另外两个用于把字符串转换成数值.

    • 传入数字 null undefined 或布尔类型时

      1. alert(Number(25)); // 25,数值直接返回  
      2. alert(Number(true)); // 1
      3. alert(Number(null); // 0,空对象返回0
      4. alert(Number(undefined)); // NaN,undefined返回NaN

    • 如果传入的是一个数组

      1. 如果数组里面只有一个数据，并且这个数据是一个数字或者一个纯数字的字符串都可以被转换

        1. Number(['0123']) // 123

      2. 如果有多个数据就是NaN

        1. Number([1,2]) // NaN

      3. 如果是空数组会被转换成0

        1. Number([]) // 0

    • 如果是字符串,应该遵循以下规划:

      1. 只包含数值的字符串,会直接转成十进制数值,如果包含前导0,即自动去掉.

        1. alert(Number("456")); // 456
        2. alert(Number("070")); // 70

      2. 只包含浮点数值的字符串,直接转成浮点数值,如果包含前导后导0,自动去掉.

        1. alert(Number("08.90")); // 8.9

      3. 字符串是空,直接返回0

        1. alert(Number("")); // 0

      4. 如果不是以上3种字符串类型,则返回NaN 

        1. alert(Number("canlan123")); // NaN

      5. 如果是对象,首先会调用valueOf()方法,然后确定返回值是否能转换成数值.如果转换结果是NaN,则基于这个返回值再调用toString()方法,再测试返回值.

        1. var box = {
        2.      toString:function(){
        3.           return 123
        4.      }
        5. }  // 123
        7. var box = {
        8.      toString:function(){
        9.           return "456"
        10.      }
        11. }  // 456
        13. var box = {
        14.      toString:function(){
        15.           return "canlan11"
        16.      }
        17. }  // NaN

  • parseInt()
    由于Number()函数在转换字符串时比较复杂且不够合理,因此在处理整数的时候更需要使用的 是parseInt().

    1. alert(parseInt("456canlan")); // 456 会返回整数部分
    2. alert(parseInt("canl456canl")); // NaN 如果第一个不是数值,返回NaN
    3. alert(parseInt("12canl")); // 12,从第一个数开始取,知道最后一个连续数值结束
    4. alert(parseInt("56.12")); // 56, 小数点不是数值,会被去掉
    5. alert(parseInt("")); // NaN,空返回NaN
    7. // parseInt()处了能够识别十进制数值,也可以识别八进制和十六进制.
    8. alert(parseInt("0xA")); // 10,十六进制
    9. alert(parseInt("070")); // 56,八进制
    10. alert(parseInt("0xAcanlan")); // 100,十六进制,canlan被自动过滤掉
    12. // ECMAScript为parseInt()提供了第二个参数,用于解决各种进制的转换.
    13. alert(parseInt("0xAF")); // 175,十六进制
    14. alert(parseInt("AF",16)); // 175,十六进制
    15. alert(parseInt("AF")); // NaN, AF不是数字
    16. alert(parseInt("101010101",2)); // 314,二进制转换
    17. alert(parseInt("70",8)); // 56,八进制转换
    19. //小数转化整数
    20. ~~a == a>>0;

  • parseFloat()
    parseFloat()是用于浮点数值之间的转换,和parseInt()一样,从第一位解析到非浮点数值位置.

    1. alert(parseFloat("123canlan")); // 123,去掉不识别的部分
    2. alert(parseFloat("0xA")); // 0,不认识十六进制
    3. alert(parseFloat("123.4.5")); // 123.4,只认一个小数点
    4. alert(parseFloat("0123.400")); // 123.4,去掉前后导
    5. alert(parseFloat("1.234e7")); // 12340000,把科学计数法转换成普通数值.

    parseInt()和parseFloat对''\null\undifined{}[]\fn\boolean都返回NaN

函数

函数参数
- 形参：就是函数定义时候的参数
- 实参：就是函数执行时候传入的参数
函数的形参和实参不一定是一一对应的，在调用函数的时候可以传入任意多个实参。

函数的属性
- fn.length: 代表函数形参的个数
- arguments: 函数内部的一个属性，它的类型是一个类数组，它的作用就是用来保存实参的集合。
- length：就代表实参的个数。

关于arguments

 函数之所以可以传入任意多个实参，实际上最终保存的地方都在这个类数组中。

做比较时注意

  1. 如果等号两边有一边式布尔值，那么会被隐式的转换为数字，false会被转换为0， true会被转换为 1。
  2. 如果是字符串数字和数字做比较的时候，那么字符串会被转换为数字做比较。
  3. 如果是 undefined 和 null 做比较的时候，返回true， 不会发生类型转换。
  4. 如果是对象在做比较的时候，需要看比较的对象的引用是否相同。
